Java BluePrints: A Member of the GlassFish Community

Welcome to the Java BluePrints project, the open development site for the Java BluePrints program from Sun Microsystems. The Java BluePrints projects presents the programming model, guidelines and examples for designing enterprise quality applications and web services using Java technologies. Some of the areas covered are Ajax-enabled Web 2.0 applications, Persistence,  JavaServer Faces, SOA with BPEL, and WS-Security.
